# [HumHub](https://www.humhub.com/) Effects Module

Add dynamic visual effects like **Snowfall**, **Rain Fall**, **Leaf Fall**, **Sakura Fall**, **Pumpkin Fall**, **Fireworks**, **Confetti**, and **Hearts** to your HumHub instance.

### Features:
- **Snowfall Effect**: Adds a snow effect.
- **Sakura Fall Effect**: Adds sakura petals falling effect.
- **Rain Fall Effect**: Adds rain falling effect.
- **Leaf Fall Effect**: Adds leaf falling effect.
- **Pumpkin Fall Effect**: Adds seasonal falling pumpkins.
- **Fireworks Effect**: Adds animated celebratory fireworks.
- **Confetti Effect**: Adds colorful confetti bursts.
- **Hearts Effect**: Adds falling hearts animation.
- **Automatic Discovery**: New `.js` effects in `resources/js` are detected automatically.
- **Admin Settings**: Toggle effects on/off from the module settings.

### Configuration
- **Enable Effects**: Toggle module effects globally.
- **Select Effect**: Choose one active effect type.

### Add New Effects
1. Add a new effect script in `resources/js`, for example `myneweffect.js`.
2. Clear HumHub assets/cache if needed.
3. Open module settings and select the newly discovered effect from the dropdown.
